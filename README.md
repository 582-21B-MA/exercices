# Exercices

Vous trouverez dans ce dépôt les exercices du cours. Les exercices ne
seront jamais évalués. Il n'est donc pas nécessaire de les remettre.

Tous les exercices doivent être faits en équipe de deux. Lorsque vous
travaillez en binôme, une personne est le ou la *conductrice*, tandis
que l'autre est le ou la *navigatrice*. Le ou la conductrice est celui
ou celle qui écrit le code. Il ou elle doit donc partager son écran.
Pendant que le ou la conductrice écrit le code, le ou la navigatrice
propose des idées, et repère les erreurs potentielles.
